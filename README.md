由于市面上不存在一个气泡脑图工具符合我的使用需求自己做了一个，市面上的脑图工具都比较拖沓做起来摆设所消耗的时间远超实际应用，针对这些做了一个极简工具。
whimsical→画气泡和连线过于麻烦
常规流程决策工具→画出来的气泡太大导致完全无法方便的用来做决策
市面上所有的脑图工具→气泡太大、操作繁琐、各有各的麻烦、流程浪费时间而且严重多余

很多时候我只需要一个决策流程列出可能性将脑海中过量储备的信息关键词整理起来，方便可视化和延时做决策并在需要时可以一眼想起来之前的进度和继续完成思索。我的意图是在最少的步数以内实现它。




















由于考虑到将来AI越来越强，和便于其他人做一套自己的工具出来，以下是耗费了蛮长时间自己慢慢琢磨出来的工程提示词。















帮我生成一个Python代码工具方便我很快很容易的能输入并且做好行动与决策流程图 
然后我的输入我可以自由拖拽拼在一起成带箭头的流程图
流程图还必须要极其紧凑便于查看，我用不了whimsical的核心原因就是它生成的流程图过于疏散我需要翻很多页才能看完整个流程，一个少于一句话的事情反而变得麻烦了很多


我想要的效果是这样的
代码会是一个窗口，窗口里都是可以自由发挥的画板
我可以在任意空的地方单击然后生成一个小长方椭圆（看起来有点像是生命条，里边是空的）然后自动进入输入状态，也就是往这个椭圆里面键入内容，它会自适应我的输入内容长度
右键这个气泡会从里面延展出箭头导向的线路我可以连上任意气泡。
我可以左键单击选中点击链接 线路重新连上其他气泡或者选中的状态下按下del删掉这些线路并且这个操作不会和单击生成新气泡混合在一起
   选中气泡的时候如果我输入del就可以删掉该气泡
随着气泡连接的数量越来越多，越是末端的气泡背景色会逐步渐变从白色到淡红到越来越红，达到10个链接时是最红的，按照这样来计划渐变色的变化程度吧

自动消除没有输入内容的空气泡。右键将两个气泡关联在一起之后会有一个永久显示的虚线线条在两个气泡间且是恒定显示的且为2px黑色，有很多条线重叠时自动变更线的色彩为{黑、红、绿、蓝、紫}以便于辨识。
所有编辑内容会自动缓存在同目录下的<workflow.json>每分钟自动保存。每次启动代码的时候自动加载这个为默认workflow。
点击选中了一个气泡或是正在编辑这个气泡时如果按下tab键，就在直右侧10px距离处生成一个新的气泡。如果按下enter就在当前气泡垂直下方10px处生成一个新的气泡。
对气泡增加文字内容编辑总是使之向右侧延展。
右键双击气泡可以编辑注释，鼠标飘过气泡右上角会在附近屏幕上任意空处显示这些注释内容直至鼠标离开该气泡。该功能有别于右键单击气泡做关联线。

移动气泡的逻辑改成，仅在单击选中了此气泡时再进行拖拽才能移动，而不是直接拖拽就能移动，直接左键拖拽没有效果。
新增一个功能，ctrl+z恢复上一个操作，仅记忆最近的三个动作。
ctrl+左键按住拖拽等于方框多选气泡，便于批量移动或者删除等。
设置排版辅助功能，snap，上下气泡沿左边对齐，左右气泡沿中线对齐，以及和左边最近的气泡距离以5px的倍数做snap对齐（控制距离的时候方便保持整齐），存在一个snap效应辅助对齐，但是不要太强以至于我有意想要不对齐会被系统强制牵引导致不理想的结果产生。
